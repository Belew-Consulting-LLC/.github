# Belew Consulting LLC

**Custom Software & AI Solutions — Built in Kansas**

We design and ship production software across the full stack: web platforms, AI agents, mobile apps, and LLM integrations. We work with a select group of clients at this time.

## What We Build

**Web Applications**
Full-stack platforms built on Python (FastAPI, Django) with Next.js/TypeScript frontends, async PostgreSQL, Redis, and Docker. From SaaS products to internal business tools.

**AI Agents & LLM Integrations**
Custom AI agents with skill systems, MCP servers, RAG pipelines with vector search (pgvector), and integrations across multiple LLM providers. We build the infrastructure that makes AI actually useful in production — credential vaults, context management, scheduling, and multi-channel delivery via Discord, Slack, and SMS. Custom skills include deployment automation, compliance scanning, and code quality and security analysis.

**AI-Powered SaaS**
End-to-end SaaS products with Stripe billing, multi-factor auth, S3 storage, transactional email, and production-grade deployment pipelines. This includes customer-facing management portals for AI employee infrastructure — provisioning and controlling Proxmox-backed virtual environments through a purpose-built web interface.

**Mobile**
Native iOS applications in Swift, including mobile companions for existing web platforms.
